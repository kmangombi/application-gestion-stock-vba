# Application de gestion de stock – Excel / VBA

## Contexte

Ce projet a été réalisé dans le cadre d’un travail universitaire en gestion de projet Agile.  
L’objectif était de concevoir une application Excel/VBA permettant de structurer, sécuriser et automatiser la gestion d’une base de données pour un concessionnaire automobile.

L’entreprise utilisait auparavant plusieurs fichiers Excel indépendants, rendant la gestion du stock complexe, peu sécurisée et difficile à analyser sur le long terme.

---

## Fonctionnalités

L’application développée permet :

- Structuration d’une base de données (véhicules, clients, transactions)
- Ajout, modification et suppression sécurisée des données
- Visualisation simplifiée des informations (stock, ventes, achats)
- Recherche multicritère (modèle, prix, ancien propriétaire…)
- Contrôle automatique de cohérence des données saisies
- Sécurisation de la saisie via formulaires VBA (pas d’accès direct aux cellules)
- Aide à l’analyse des performances d’achat (optimisation du stock)

---

## RGPD & Sécurisation des données

Une réflexion spécifique a été menée sur la conformité RGPD :

- Identification des données personnelles collectées
- Définition des données à conserver / archiver
- Limitation des accès utilisateurs
- Sécurisation des formulaires de saisie
- Documentation des choix techniques

Un rapport RGPD a été rédigé afin d’analyser la conformité du projet aux principes de protection des données.

---

## Méthode Agile

Le projet a été conduit selon la méthodologie Scrum :

- Organisation en 2 sprints
- Gestion des tâches via Trello
- Création d’un Product Backlog
- Rédaction de comptes rendus de sprint
- Revue avec représentant du client
- Rétrospectives d’équipe

Cette approche a permis :

- Une meilleure organisation du travail
- Une répartition claire des tâches
- Une amélioration continue du projet
